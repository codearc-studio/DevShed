# DevShed

### The cleanest way to manage your Xcode storage.

![DevShed Header](INSERT_HEADER_IMAGE_URL_HERE)

DevShed is a high-performance, native macOS utility built for developers who need to keep their workstations lean. Xcode is notorious for hoarding gigabytes of hidden junk; DevShed puts that space back in your hands with a native, one-click interface.

---

## Features

*   **Surgical Cleaning:** Selectively wipe Derived Data, Archives, Simulator diagnostics, Device Support binaries, and Xcode Caches.
*   **One-Click Purge All:** A smart, safe master-purge that recovers maximum space without touching your important project metadata.
*   **Lifetime Metrics:** Watch your total reclaimed storage grow in real-time.
*   **Custom Watchlist:** Have a folder that clutters your drive? Add any directory to DevShed’s custom monitor and clear it on demand.
*   **Health Alerts:** Get notified the moment your Xcode storage threshold is exceeded.

## Getting Started

1.  **Download:** Grab the latest .zip release from the [Releases Page](https://github.com/YOUR_GITHUB_USERNAME/DevShed/releases).
2.  **Install:** Move DevShed.app to your Applications folder.
3.  **Launch:** Open the app and find the shipping box icon in your menu bar.
4.  **Clean:** Click to see your storage breakdown and reclaim your disk space.

*Note: If you receive a security prompt, right-click the app in Finder, select Open, and confirm the security dialog.*

---

## Built for Pro Workflows

DevShed is designed to be as lightweight as possible. It is built using:

*   Swift and SwiftUI: Fully native UI for perfect performance.
*   AppKit: Deep macOS system integration.
*   Concurrency: Parallelized scanning processes so your Mac stays fast while we find the junk.
